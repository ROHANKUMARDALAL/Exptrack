# 📱 Expense Tracker

A modern Expense Tracker application built with React Native and Node.js that helps users manage personal and group expenses efficiently.

---

## ✨ Current Features

### Authentication

- User Login
- User Registration
- Token Verification using AsyncStorage
- Auto Login (Session Persistence)
- Logout

### Expense Management

- Add New Expense
- Add Expense Description
- Track All Expenses
- Search Expenses
- Filter Expenses by Date
- View Expense History

### Group Management

- Create Group / Project
- Add New Members
- Track Member-wise Expenses
- View "Added By" information for every expense

### State Management

- Redux Toolkit for Global State Management
